# /word-list コマンド

単語帳に登録されている全単語を一覧表示するコマンドです。

## 手順

1. `~/.word-glossary/glossary.json` を読み込んでください（`~` はユーザーのホームディレクトリ）。
   - ファイルが存在しない場合は「📖 単語帳はまだ作成されていません。`/word-add <単語>` で単語を追加してください。」と表示して終了してください。

2. `words` 配列が空（0件）の場合は「📖 単語帳は空です。`/word-add <単語>` で単語を追加してください。」と表示して終了してください。

3. `words` 配列に単語がある場合は、以下の形式で一覧表示してください：

   ```
   📖 単語帳（<件数>件）
   ---
   1. <term> — <meaning の先頭部分（長い場合は省略して「…」を付ける）>
   2. <term> — <meaning>
   3. <term> — <meaning>
   ...
   ```

   - 番号は 1 から順に振ってください。
   - `addedAt` の日時順（古い順）で表示してください。
   - 意味の説明が長い場合は、50文字程度で省略して末尾に「…」を付けてください。
