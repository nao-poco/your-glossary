# /word-add コマンド

単語帳に新しい単語を追加するコマンドです。

## 引数

$ARGUMENTS に追加したい単語が入ります。

## 手順

1. `$ARGUMENTS` が空の場合は「追加したい単語を指定してください。例: /word-add decorator」と表示して終了してください。

2. プロジェクトルートの `.word-glossary/glossary.json` を読み込んでください。
   - ファイルが存在しない場合は、`.word-glossary/` ディレクトリごと作成し、以下の初期データを用意してください：
     ```json
     {
       "version": "1.0",
       "words": []
     }
     ```

3. `words` 配列の中に、同じ `term` の単語が既に存在するか確認してください（大文字小文字は区別しない）。
   - 既に存在する場合：「⚠️ "$ARGUMENTS" は既に登録されています。上書きしますか？」とユーザーに確認してください。
     - ユーザーが上書きを希望した場合は、既存のエントリを削除してから次のステップへ進んでください。
     - ユーザーが上書きを希望しない場合は、処理を中止してください。

4. `$ARGUMENTS` の単語について、プログラミング・IT の文脈での意味を生成してください。
   - ユーザーの使用言語に合わせて回答してください（日本語でやり取りしている場合は日本語で説明）。
   - 説明は簡潔に、2〜3文程度にしてください。

5. 以下の形式で `words` 配列に新しいエントリを追加してください：
   ```json
   {
     "term": "$ARGUMENTS の値（原文のまま）",
     "meaning": "生成した意味の説明",
     "addedAt": "現在の日時（ISO 8601形式）"
   }
   ```

6. 更新した JSON を `.word-glossary/glossary.json` に保存してください。

7. 以下の形式で結果を表示してください：
   ```
   ✅ "<単語>" を追加しました
   ---
   <単語>（読み方があれば）: <生成した意味の説明>
   ```
