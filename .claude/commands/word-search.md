# /word-search コマンド

単語帳からキーワードで単語を検索するコマンドです。

## 引数

$ARGUMENTS に検索キーワードが入ります。

## 手順

1. `$ARGUMENTS` が空の場合は「検索キーワードを指定してください。例: /word-search function」と表示して終了してください。

2. プロジェクトルートの `.your-glossary/glossary.json` を読み込んでください。
   - ファイルが存在しない場合は「📖 単語帳がまだ作成されていません。`/word-add <単語>` で単語を追加してください。」と表示して終了してください。

3. `words` 配列の中から、以下の条件に一致するエントリをすべて検索してください（大文字小文字は区別しない）：
   - `term` にキーワードが含まれる
   - `meaning` にキーワードが含まれる

4. 検索結果が0件の場合は「🔍 "<$ARGUMENTS>" に一致する単語は見つかりませんでした。」と表示して終了してください。

5. 検索結果がある場合は、以下の形式で表示してください：

   ```
   🔍 "<キーワード>" の検索結果（<件数>件）
   ---
   1. <term> — <meaning>
   2. <term> — <meaning>
   ...
   ```

   - 番号は 1 から順に振ってください。
   - 意味の説明は省略せず全文を表示してください。
